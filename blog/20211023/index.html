Spring Boot部署至伺服器，卻無法顯示頁面

By Sherry 2021-10-22 Spring boot
在本機開發測試都能正常顯示網頁，但當專案匯出war部署至tomcat上，網頁卻顯示一片空白…
解決方式:
  1. 開啟pom.xml檔案：檢查是否在<properties>有設定主類位置 <start-class>com.xxx.xxxApplication</start-class>
  2. 檢查頁面放置路徑
  3. 專案 > Properties > Deployment Assembly > 檢查是否Source有指定頁面放置路徑，沒有則Add
  4. 重新匯出war 再次部署至tomcat上即可
